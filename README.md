## 菜鸟一枚, 网站只是为了学习/进步
## 本网站使用Hexo, 基于主题 [hexo-theme-icarus](https://github.com/ppoffice/hexo-theme-icarus)

## 上传github流程

### 第一步
npm install hexo-cli -g

```
hexo clear
```

### 第二步

```
hexo d -g
```

### 本地运行测试 http://localhost:4000

```
hexo s
```
